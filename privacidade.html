Política de Privacidade

Sua privacidade, respeitada por design.

Última atualização: 2026

O aplicativo Tennis Custom respeita a sua privacidade e está comprometido em proteger seus dados.

01Coleta de dados

O Tennis Custom não coleta dados pessoais identificáveis dos usuários.

O aplicativo pode armazenar informações localmente no dispositivo, como configurações de raquetes e preferências do usuário, exclusivamente para funcionamento interno.

02Compras dentro do app

As compras realizadas no aplicativo são processadas pela Apple (App Store). Não temos acesso a informações de pagamento, dados bancários ou dados pessoais relacionados às transações.

03Uso de dados

Como não coletamos dados pessoais, nenhuma informação é compartilhada, vendida ou utilizada para fins de marketing.

04Armazenamento de dados

Todas as informações do usuário ficam armazenadas localmente no dispositivo e não são enviadas para servidores externos.

05Segurança

O aplicativo não utiliza armazenamento em nuvem ou coleta de dados sensíveis, reduzindo riscos de exposição.

06Privacidade de crianças

O aplicativo não coleta informações de crianças ou usuários de qualquer idade.

07Alterações nesta política

Esta política pode ser atualizada ocasionalmente. Recomendamos revisar esta página periodicamente.
